## Windows添加静态路由
>2019/07/11
#### 添加永久静态路由
- 添加一条永久静态路由
```bash
$route ADD -p 192.168.1.1 MASK 255.255.255.0 192.168.1.1 METRIC 3 IF 2
            destination^      ^mask           ^gateway   metric^    ^
                                                           Interface^
```
- 查看路由表
```bash
$route print
IPv4 路由表
===========================================================================

网络目标        网络掩码          网关       接口   跃点数
          0.0.0.0          0.0.0.0        10.30.1.1       10.30.1.88     15
        10.30.1.0    255.255.255.0            在链路上        10.30.1.88    271
       10.30.1.88  255.255.255.255            在链路上        10.30.1.88    271
      10.30.1.255  255.255.255.255            在链路上        10.30.1.88    271

永久路由:
  无
```
- 删除一条路由表
```bash
route delete 192.168.100.0 #如果有两条路由记录有着相同的“目的网络号”，则会将两条记录同时删除。如果只想删除其中某一条的话，请在删除后用route add命令再添加上其中一条。
```
- 修改一条路由记录
```bash
route change 157.0.0.0 mask 255.0.0.0 157.55.80.5 metric 2 if 2 #CHANGE参数只能修改“网关”和“跳数”。
```
